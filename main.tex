\subsection*{2.3. Pra-pemrosesan Data (\textit{Data Preprocessing})}
Sebelum dilatih menggunakan model \textit{Multilayer Perceptron} (MLP), data diproses melalui langkah-langkah berikut:
\begin{itemize}
    \item \textbf{\textit{Variabel Encoding}:} Target diubah ke dalam bentuk numerik, yaitu \texttt{'M'} menjadi 1 dan \texttt{'B'} menjadi 0.
    \item \textbf{Pemisahan Data:} Perbandingan 75\%:25\%.
    \item \textbf{Penskalaan Fitur (\textit{Feature Scaling}):} Karena nilai-nilai fitur memiliki rentang skala yang berbeda-beda, dilakukan standardisasi menggunakan \texttt{StandardScaler} agar nilai fitur berpusat di rata-rata 0 dengan variansi 1.
\end{itemize}

\subsection*{2.4. Arsitektur Model dan Hyperparameter}

Eksperimen klasifikasi dilakukan menggunakan \texttt{MLPClassifier} dari pustaka \textit{scikit-learn}. Sebagai langkah awal, pengujian menggunakan \textit{basemodel} (tanpa modifikasi parameter dari dokumentasi bawaan \textit{scikit-learn}) berhasil memperoleh akurasi sebesar $0.986013986013986$. Berdasarkan eksperimen \textit{tuning} yang dilakukan setelahnya, diperoleh arsitektur dan \textit{hyperparameter} terbaik untuk dataset ini. Konfigurasi tersebut menggunakan konfigurasi sebagai berikut.
\begin{verbatim}
    mlp_clf_tuned = MLPClassifier(
    hidden_layer_sizes=(100,),
    alpha=1e-2,
    activation='logistic',
    solver='adam',
    random_state=42
)
\end{verbatim}

\subsection*{3.2. Analisis Eksperimen}
Berdasarkan hasil akurasi sebesar 99.3\%, model MLP terbukti sangat efektif untuk tugas klasifikasi pada dataset \textit{Breast Cancer Wisconsin}.
Penggunaan \textit{Feature Scaling} dengan \texttt{StandardScaler} memainkan peran penting di sini, karena algoritma berbasis jaringan saraf (seperti \textit{solver} \texttt{adam}) sangat sensitif terhadap skala fitur. Perbedaan skala data yang mencolok dapat memperlambat laju konvergensi dan menghasilkan performa yang suboptimal.

Penerapan nilai regularisasi L2 (\texttt{alpha=0.01}) juga membantu menjaga agar bobot model tidak menjadi terlalu besar, sehingga meminimalisir kemungkinan terjadinya \textit{overfitting} meskipun model memiliki 100 neuron pada lapisan tersembunyinya. Fungsi aktivasi \texttt{logistic} sudah cukup mapan menangani data yang skalanya dinormalisasi, menghasilkan prediksi diagnosis yang sangat presisi dalam membedakan tumor jinak dan ganas.
